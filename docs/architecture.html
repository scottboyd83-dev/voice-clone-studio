System Architecture

How the Studio works

A React control surface, one FastAPI core, and three isolated ML engines — each in its own Python environment so their dependencies never collide. Everything below the browser runs on this machine; no audio, text or model ever leaves it.

Three journeys through the stack

Generate text + voice▸ app.py▸ F5-TTS or GPT-SoVITS▸ wav + seed▸ generations/ + History Instant voices synthesize in-process; ★ fine-tuned voices proxy to the GPT-SoVITS api_v2 server with top-k / top-p / temperature.
Train 90 prompts▸ quality gate▸ Whisper verify▸ s2 + s1 training▸ ★ voice in library Takes are loudness-normalised, transcript-checked (score < 0.6 excluded), then trained overnight on CPU in the isolated venv.
Convert your performance▸ app.py▸ Seed-VC worker▸ same words, new timbre▸ conversions/ Target = any library voice's 10s reference clip. Pacing and emotion survive; singing mode adds pitch tracking at 44.1 kHz.
